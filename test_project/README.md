# Test Project

This is a test project for demonstrating multiple file and folder upload functionality.

## Structure

- `main.py` - Main application entry point
- `utils.py` - Utility functions
- `config.py` - Configuration settings
- `submodule/` - Submodule directory
  - `__init__.py` - Module initialization
  - `database.py` - Database functionality
  - `api.py` - API client

## Usage

Run the main application:
```bash
python main.py
```

This will demonstrate the multi-file project structure.
